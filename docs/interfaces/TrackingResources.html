      ): Promise<
          { action: string; category: string; count: string; storageDate: string }[],
      >

      Retrieves event counters aggregated by action for a given category.

      Parameters

      • category: string

        The tracking category name to query (e.g. "payments").

      • params: GetEventCountersParams

        Query parameters including date filters and optional pagination.

        • filters

          Required date range filters (startDate and endDate).

        • pagination

          Optional pagination controls (skip and take). Defaults to skip=0, take=100.

      Returns Promise<
          { action: string; category: string; count: string; storageDate: string }[],
      >

      A promise that resolves to an array of EventCounter objects.

      Example

      const counters = await client.trackings.getCategoryCounters("payments", {
        filters: { startDate: "2024-01-01", endDate: "2024-01-31" },
        pagination: { skip: 0, take: 50 },
      });
      // => [{ category: "payments", action: "success-order", count: "4", storageDate: "..." }]
